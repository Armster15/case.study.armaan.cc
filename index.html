How it Works

Well, it's pretty simple. You know what 404 pages are, right?
If you don't they are pages that tell you that a page doesn't exist.

Here are a few 404 page examples

  • https://www.apple.com/This-Page-Does-Not-Exist
  • https://www.microsoft.com/This-Page-Does-Not-Exist
  • https://www.armaan.cc/This-Page-Does-Not-Exist


So these pages redirect you to a page in which it tells you that page doesn't exist.

This website has a 404 page as well. But while the websites above have a proper 404 page, this webpage's 404 page redirects you to Rick Astley's Never Gonna Give You Up. If you are curious what the 404 file is called, its called rickroll.php and can be accessed at http://case.study.armaan.cc/rickroll.php. If you want to view the code of the 404 page, see Nerd Land.
